The Borders CSS Box Model

In this CSS box model, a defined max-width constrains the content width of the heading, and a defined min-height constrains the box height so that it's taller than the rendered text.

Using CSS, I can style each side of the border area of this blockquote to be unique. The border-right, border-left, border-top, and border-bottom properties are all defined with three values: the first defines its width, the second its style, and the third its color.
